# ColorPencil

You can change the color of terminal text.

## Installation

Add this line to your application's Gemfile:

```
gem 'color_pencil'
```

And then execute:

```
$ bundle
```

## Usage

```ruby:
color_pencil = ColorPencil.new
color_pencil.red 'foo'
color_pencil.blue 'bar'
color_pencil.green 'baz'
```
